# The pins connected to the Logic Analyser

| LOGIC CHx | Board Pin | Function | Color  |
| --------- | --------- | -------- | ------ |
| CH1       | A2        | MOSI     | White  |
| CH2       | A3        | MISO     | Brown  |
| CH3       | C6        | Clock    | Red    |
| CH4       | C5        | Enable   | Orange |
| GND       | GND       | GND      | Black  |
